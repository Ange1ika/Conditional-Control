## Kepler Py — лабораторный проект

Проект предназначен для реализации «перевода Кеплера» — преобразования кеплеровых элементов орбиты
в радиус‑вектор и вектор скорости аппарата в инерциальной системе координат.

### Структура

- `src/kepler/` — исходники библиотеки:
  - `constants.py` — физические константы.
  - `kepler_orbit.py` — функции для работы с орбитой (перевод Кеплера).
  - `main.py` — простой пример использования из командной строки.
- `tests/` — базовые тесты для проверки корректности реализации.

### Быстрый старт

```bash
cd kepler_py
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

python -m kepler.main
```
